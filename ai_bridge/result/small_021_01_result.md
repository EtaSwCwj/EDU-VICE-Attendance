## 🌐 테스트 환경 구성

### 멀티 계정 테스트
| 창 | 모드 | URL | 용도 |
|---|---|---|---|
| 1 | 일반 | localhost:8080 | 계정 A |
| 2 | 시크릿 | localhost:8080 | 계정 B |
| 3 | 모바일 | 앱 실행 | 카메라/사진 |

### 테스트 시나리오
- **관리자** vs **선생님**: 동시 접속
- **선생님** vs **학생**: 실시간 출석체크
- **학부모** vs **학생**: 출석 현황 확인

## 🚀 실행 방법

1. **웹 서버 시작**
   ```bash
   cd C:\gitproject\EDU-VICE-Attendance\scripts
   .\run_web_test.bat
   ```

2. **첫 번째 창** (일반 모드)
   - 자동으로 Chrome이 열림
   - localhost:8080 접속

3. **두 번째 창** (시크릿 모드)
   - Ctrl+Shift+N으로 시크릿 창 열기
   - localhost:8080 접속
   - 다른 계정으로 로그인

## ⚠️ 주의사항

1. **포트 충돌**: 8080 포트 사용 중이면 배치 파일에서 포트 변경
2. **CORS 에러**: --disable-web-security 플래그로 해결
3. **캐시 문제**: 시크릿 모드로 세션 분리
